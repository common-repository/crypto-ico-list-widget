<h2>Special Note About ICO List Database and API</h2>
This Crypto ICO List Widget plugin totally depends on a third party ico-list data api - 'https://us-central1-ico-data.cloudfunctions.net/icolist/'

CoolPlugins.net, Our plugin development team regularly updates active and upcoming ICO details inside this ICO list database API from where you can directly access it on your website with automatic API call inside the plugin. We grab active crypto ICOs information from various internet information resources - twitter, facebook, PR and news announcements by ICO owners etc.
